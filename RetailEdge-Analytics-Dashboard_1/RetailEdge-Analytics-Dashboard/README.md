## Dataset Overview

| Field | Description |
|-------|-------------|
| InvoiceNo | Unique invoice identifier |
| InvoiceDate | Transaction date (2023-01-01 to 2023-12-31) |
| CustomerID | Customer code (GUEST for unregistered) |
| CustomerName | Business/customer name |
| Country | 15 countries including UK, Germany, France, etc. |
| StockCode | Product SKU |
| Description | Product name |
| Category | Product category (7 categories) |
| Quantity | Units sold per line |
| UnitPrice | Selling price per unit |
| Discount | Discount rate applied (0 to 0.15) |
| LineTotal | Net revenue for the line |
| CostPrice | Cost of goods for the line |

**Dataset stats:** 44,787 rows ¬∑ 15 countries ¬∑ 20 products ¬∑ 20 named customers

## Quick Start

### 1. Clone & install dependencies

```bash
git clone https://github.com/YOUR-USERNAME/RetailEdge-Analytics-Dashboard.git
cd RetailEdge-Analytics-Dashboard
pip install -r requirements.txt
```

### 2. Run the data cleaning pipeline

```bash
python scripts/data_cleaning.py
```

Output:
```
[INFO] Loaded 44,787 rows
[INFO] Removed 53 duplicate rows
[INFO] Cleaned data saved to 'data/processed/retail_sales_cleaned.csv'
```

### 3. Run the business analysis

```bash
python scripts/data_analysis.py
```

Produces 8 summary CSV files in `/outputs/` including RFM segmentation.

### 4. Generate all charts

```bash
python scripts/visualizations.py
```

Produces 7 dashboard-quality PNG charts in `/outputs/`.

### 5. Launch the Jupyter notebook

```bash
jupyter notebook notebooks/exploratory_analysis.ipynb
```

---

## SQL Usage

Load schema into PostgreSQL:

```bash
psql -U postgres -d retailedge -f sql/schema.sql
psql -U postgres -d retailedge -f sql/views.sql
```

Run BI queries:

```bash
psql -U postgres -d retailedge -f sql/queries.sql
```

## Key Findings

- **Q4 dominates:** Q4 accounted for ~32% of full-year revenue, driven by Nov‚ÄìDec demand spikes
- **Germany is the #1 international market**, generating over ¬£500k outside the UK
- **Home Decor leads all categories** at 40% revenue share
- **Top 5 customers** each exceeded ¬£260k in annual spend ‚Äî strong B2B concentration risk
- **Himalayan Salt Lamp** was the single best-selling product by revenue
- **RFM analysis** revealed 6 Champion customers and 7 Potential Loyalists to nurture
